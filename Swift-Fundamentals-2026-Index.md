# Swift Fundamentals 2026

This is a day-wise Swift interview preparation track from fundamentals to modern Swift 6.x topics.

## How To Use

1. Start with `00-day-wise-roadmap.md` to understand the full preparation path.
2. Study one day at a time.
3. Each topic has its own Markdown file with:
   - Core explanation
   - Practical examples
   - iOS use cases
   - Swift 6.1, 6.2, and 6.3 notes where relevant
   - Quick interview notes
   - Points to remember
   - Practice questions

## Current Version Note

As of July 16, 2026, Swift.org shows Swift 6.3.3 as the current install version. Swift 6.3 has an official release post. I could not verify an official Swift 6.4 release from Swift.org, so this module covers verified Swift 6.1, 6.2, and 6.3 changes and leaves room to add Swift 6.4 once it is officially published.
